# Analog-Clock
This project is a simple analog clock website built using HTML, CSS, and JavaScript. It provides users with simple time telling.

# Features

- **analog-clock:** Display time with moving hands and hours with marking from 1 to 12 hours to show the time.
- **Search:** Ability for basic time telling.

# Technologies Used

- **HTML:** Used for structuring the webpage and its content.
- **CSS:** Styled the website to make it visually appealing and user-friendly.
- **JavaScript:** Implemented functionality for fetching math data and dynamically updating the webpage.
